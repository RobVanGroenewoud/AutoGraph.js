AutoGraph.js

Examples

AutoGraph.js supports multiple ways to define graphs. This examples page shows the various options.

  • Minimal
  • Advanced
  • Error

Minimal Default configuration

Code

<figure data-graph="data1.xml">
</figure>
Show data file

Advanced Additional settings

Code

<figure 
data-graph="data2.xml" 
data-graph-title="Insert graph title here">
</figure>
Show data file

Error In case of missing or invalid configuration

Code

<figure data-graph="nonexisting.xml">
</figure>
